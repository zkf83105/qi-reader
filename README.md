## 用起来是什么感觉？

界面很简洁，第一眼没有"哇"的那种惊艳，但用下去你会觉得舒服——像一间收拾整齐的房间，东西都在该在的地方。

几个让我印象深的功能：

**内嵌浏览器 + 全文加载**：这是最实用的一个。很多 RSS 源只输出摘要，要看全文得跳转到原网站，然后遇上一堆广告弹窗。Qi Reader 的全文输出功能可以直接在阅读器里拉取完整文章，内嵌浏览器还能直接点赞、评论，基本不用离开这个界面。

**划词翻译 + 英汉词典**：追一些英文技术博客或者海外资讯的话，这个功能很实在。选中单词直接出释义，不用切换 App。

**稍后读 + 标签分类**：碰到好文章来不及看，一键收藏，打上标签，之后按标签找。搞研究、做知识管理的用户会喜欢这个。

**三种布局 + 深浅主题**：列表、杂志、三栏随便切，字体大小行高都能调。夜猫子用深色模式，完全没问题。

**PWA 安装 + 快捷键**：可以把 Qi Reader 安装成 PWA，体验上接近本地 App。配上键盘快捷键操作，翻文章、标记已读、收藏一气呵成，效率党会喜欢。

**内置中英文订阅源库**：刚上手不知道订什么？没关系，软件内置了一批精选订阅源，少数派、IT之家这类常用的中文站直接一键加，省去找源的麻烦。
